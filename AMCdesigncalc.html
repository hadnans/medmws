📌 Design Starting Logic

Rule 1 — Unit Cell Size
Start with a ≈ λ₀/10 at your target frequency. This keeps the structure sub-wavelength (a true metamaterial condition).
Rule 2 — Gap Size
Gap g = (0.1–0.2)×a. A smaller gap increases capacitance → lowers f₀. A larger gap does the opposite.
Rule 3 — Substrate Thickness
Auto-set to t ≈ λ₀/40 as a starting point. Thicker substrate → more inductance → lower f₀.

Complete HFSS Workflow — What to Do, Why, and Where

  • Create the HFSS Project & Draw the Substrate Modeler
    📌 Draw a box: dimensions a × a × t. Assign your dielectric material from the material library.
    💡 This is the unit cell base. The periodic repetition of this cell simulates the infinite AMC ground plane.
    HFSS → Draw → Box. Then: right-click the object → Assign Material → search your substrate (e.g. FR4_epoxy). If not found, create a new material with your εᵣ and loss tangent.
  • Draw the Ground Plane PEC
    📌 Draw a rectangle (a × a) at the bottom face (z = 0) of the substrate. Assign PEC.
    💡 The ground plane provides the inductance (L) in the equivalent LC model. Without it, the surface has no resonance.
    Draw → Rectangle on the Z=0 face. Then: right-click → Assign Boundary → Perfect E (PEC). Or assign a finite-conductivity copper sheet (σ = 5.8×10⁷ S/m) for more realistic loss simulation.
  • Draw the Patch PEC
    📌 Draw a square patch of width w = a − g, centered on the top face of the substrate (z = t). Assign PEC.
    💡 Adjacent patches separated by gap g form the capacitance (C) of the LC model. The patch geometry directly controls f₀.
    Draw → Rectangle on the Z=t face, centered at (a/2, a/2). Set width and height to w. Assign PEC boundary. You can also try cross-shaped, ring, or fractal patches for multiband AMC.
  • Create the Airbox (Radiation Region) Vacuum
    📌 Draw a vacuum box on top of the substrate: dimensions a × a × h, where h ≥ λ₀/4 at the lowest simulated frequency.
    💡 The airbox provides space for the incident and reflected plane waves. Too thin a box causes near-field errors in phase measurement.
    Draw a second box from (0, 0, t) to (a, a, t+h). Assign material: vacuum. This box top face will be your Floquet Port excitation surface.
  • Assign Periodic Boundary Conditions Boundaries
    📌 Assign PEC on two opposite side faces (E-field parallel), PMC on the other two (E-field perpendicular). OR use Master/Slave pairs.
    💡 These BCs make HFSS simulate one unit cell as if it were surrounded by infinite identical copies — the array environment. Without them, you're simulating a single isolated patch, not an AMC.
    Select face → Assign Boundary → Perfect E (for PEC walls) or Perfect H (for PMC walls). For Master/Slave: select Master face, then Slave face on the opposite side, link them with phase shift = 0° for normal incidence.
  • Assign Floquet Port Excitation Excitation
    📌 Select the top face of the airbox → Assign Excitation → Floquet Port. Set modes to TE and TM, de-embed by distance h (airbox height).
    💡 The Floquet port launches a plane wave that uniformly illuminates the AMC surface, exactly as a real incoming wave would. De-embedding shifts the reference plane from the top of the airbox down to the AMC surface.
    Assign Excitation → Floquet Port → set 2 modes (TE00, TM00). In "Modes" tab, enable de-embedding and enter the airbox height h as the de-embed distance. This is critical — without de-embedding your phase reference is wrong.
  • Setup Solution & Frequency Sweep Analysis
    📌 Add Solution Setup: set frequency = f₀, max passes = 15–20, delta S = 0.01. Add a Fast/Interpolating sweep from 0.3f₀ to 2f₀.
    💡 The adaptive meshing at f₀ ensures accurate results near resonance. The sweep reveals the full S-curve phase response and confirms the 0° crossing location.
    HFSS → Analysis → Add Solution Setup. Then right-click Setup → Add Frequency Sweep → Type: Interpolating, Start/Stop: your range, ~500 points. Fewer points may miss the sharp phase transition near f₀.
  • Post-Process: Plot Reflection Phase Results
    📌 After simulation, create an output variable: ang_deg(S(FloquetPort1:1, FloquetPort1:1)). Plot vs. frequency.
    💡 The reflection phase plot is the primary verification tool. The 0° crossing = resonant frequency. The ±90° crossings define the usable AMC bandwidth.
    Results → Create Modal Solution Data Report → Rectangular Plot. Y-axis: ang_deg(S[1,1]) in degrees. Look for the S-curve. If the 0° crossing is off-target, go to parametric sweep tab for correction.

Parametric Sweep — What to Vary, What to Look For

🔄 How to Set Up in HFSS

1. First, make all dimensions variables in HFSS: Design → Edit Variables (e.g., a_cell, gap, t_sub, w_patch = a_cell - gap).
2. Then: HFSS → Optimetrics → Add → Parametric.
3. Add each variable with its sweep range and step size.
4. Check "Save Fields and Mesh" only if you need field plots — it slows down the sweep.
5. After sweep: Results → Create Modal Solution Data Report → select all sweep variants → plot ang_deg(S(1,1)) vs. frequency.
Variable Suggested Sweep Range Effect on f₀ (0° crossing) Effect on BW (±90° range) What to Look For
w (patch width) 0.6a → 0.9a in steps of 0.05a ↑ w → ↓ f₀ (more capacitance) ↑ w → slightly narrower BW Find the w where 0° crossing aligns with your target f₀. This is your primary tuning knob.
g (gap) 0.05a → 0.3a in steps of 0.025a ↑ g → ↑ f₀ (less capacitance) ↑ g → wider BW Use g to fine-tune f₀ after fixing w. Larger gaps give you wider bandwidth but shift f₀ up.
t (substrate thickness) 0.5 mm → 3 mm in steps of 0.25 mm ↑ t → ↓ f₀ (more inductance) ↑ t → wider BW Thicker substrate gives more bandwidth but can excite surface waves. Look for BW maximization without degrading phase linearity.
εᵣ (permittivity) 2.2, 3.55, 4.4, 6.15, 10.2 ↑ εᵣ → ↓ f₀ (wave slows down) ↑ εᵣ → narrower BW High εᵣ compresses the structure (useful for miniaturization) but sacrifices bandwidth. Best tradeoff is typically εᵣ = 2.2 – 4.4.
a (unit cell size) 0.05λ₀ → 0.15λ₀ ↑ a → ↓ f₀ Minor effect Changing a scales everything. After fixing a, use w and g for fine-tuning. Ensure a < λ₀/5 at all times for sub-wavelength behavior.

✅ What a Good Result Looks Like

  • ✔ Phase crosses 0° exactly at your target f₀
  • ✔ Smooth S-curve — no discontinuities or kinks
  • ✔ ±90° bandwidth covers your antenna's operating band
  • ✔ |S₁₁| ≈ 0 dB (magnitude near 1.0) — the AMC is reflective, not absorptive
  • ✔ Phase is monotonically decreasing from +180° to −180°

❌ Warning Signs in the Sweep

  • ✖ Multiple 0° crossings → surface wave modes excited; reduce t or a
  • ✖ Phase jump / discontinuity → check de-embedding value is correct
  • ✖ |S₁₁| drops below −3 dB → lossy substrate absorbing energy; reduce loss tangent
  • ✖ Very narrow BW (<2%) → substrate too thin or εᵣ too high; increase t
  • ✖ 0° crossing not moving when you sweep w → check if patch variable is linked correctly

📌 How to Read the Phase Plot — Summary

+180° region (low freq)
Surface acts like a normal metallic reflector (PEC-like). Waves reflect with polarity inversion. Not suitable for low-profile antenna placement.
0° region (resonant band)
True AMC behavior — reflected wave is in-phase with incident wave. Ideal for flush-mounted antennas. AMC bandwidth = ±90° window.
−180° region (high freq)
Phase returns toward −180° (PEC-like again). The surface stops being an AMC. Antenna performance degrades rapidly outside the ±90° window.
